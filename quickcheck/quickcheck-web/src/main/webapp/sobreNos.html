SOBRE NÓS

O quickcheck é um site criado com o propósito de amenizar o principal problema de hospitais e postos atualmente, que é o tempo de espera para o atendimento. Isso é feito graças a possibilidade do usuário de realizar um pré-diagnóstico online, onde recebe um retorno sobre qual pode ser sua doença, a gravidade (incluindo o sistema universal de cores usado nos hospitais) e já é encaminhado para o tratamento no hospital mais próximo.
